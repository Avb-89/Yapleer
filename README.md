# Yapleer

Yapleer — сделанное для себя macOS-приложение для простого прослушивания Яндекс Музыки и «Моей волны».
Никаких окон на весь экран и прочей лабуды. 

## Установка

1. Скачайте `Yapleer.dmg`.
2. Откройте DMG.
3. Перетащите `Yapleer.app` в папку `Applications`.
4. Запустите Yapleer.
5. Приложение сразу появится в трее  (верхняя часть экрана где часы и меню) 

## Первый запуск

При первом запуске приложение предложит подключить Яндекс Музыку.

Нажмите «Войти через Яндекс».  
Приложение откроет браузер, где нужно подтвердить вход. После этого Yapleer сможет запускать музыку.

Для работы нужен аккаунт Яндекс Музыки.

## Если macOS не даёт открыть приложение

Yapleer пока не подписан и не notarized через Apple Developer ID, поэтому macOS может показать предупреждение, что приложение скачано из интернета или не может быть проверено.

Сначала попробуйте обычный способ:

1. Откройте `Applications`.
2. Найдите `Yapleer.app`.
3. Нажмите по нему правой кнопкой мыши.
4. Выберите `Open / Открыть`.
6. Подтвердите запуск.

Обычно это нужно сделать только один раз.

Если macOS всё равно пишет, что приложение повреждено или не может быть открыто, выполните в Terminal:

```bash
xattr -dr com.apple.quarantine /Applications/Yapleer.app
```
И потом открыть приложение обычным спообом
